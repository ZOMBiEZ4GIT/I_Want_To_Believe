# 5-MINUTE TASKS
## For Low-Energy, No-Focus Days

**Can't focus? Don't have energy? These tasks take 5 minutes or less.**

**Something > Nothing**

---

## EPISODE PLANNING (5 min each)

- [ ] Read ONE episode brief
- [ ] Browse episode list and pick your next episode
- [ ] Skim ONE full framework's table of contents
- [ ] Check ONE episode's COMPLETION-TRACKER.md status
- [ ] Decide: which episode do I want to record this week?

---

## PROGRESS TRACKING (5 min each)

- [ ] Update ONE COMPLETION-TRACKER.md with today's work
- [ ] Fill in ONE square in your progress grid
- [ ] Calculate XP earned today
- [ ] Update MOTIVATION-TRACKER.md current status
- [ ] Count how many episodes you've completed

---

## ORGANIZATION (5 min each)

- [ ] Rename ONE audio file properly
- [ ] Move ONE completed episode to archive folder
- [ ] Create folder for next episode
- [ ] Clean desktop of podcast files
- [ ] Organize downloads folder

---

## SKILL BUILDING (5 min each)

- [ ] Practice ONE difficult pronunciation
- [ ] Read ONE paragraph aloud for pacing practice
- [ ] Listen to 5 minutes of your latest episode
- [ ] Watch 5 minutes of a tutorial
- [ ] Test your microphone levels

---

## RESEARCH (5 min each)

- [ ] Bookmark ONE source for upcoming episode
- [ ] Read ONE section of an episode framework
- [ ] Google ONE fact to verify
- [ ] Find ONE image reference for episode
- [ ] Check ONE claim in the framework

---

## COMMUNITY (5 min each)

- [ ] Post ONE social media update about your progress
- [ ] Respond to ONE comment
- [ ] Read ONE podcast subreddit post
- [ ] Join ONE podcasting Discord/Slack
- [ ] Tell ONE person about your progress

---

## ADMIN (5 min each)

- [ ] Set calendar reminder for next recording session
- [ ] Write down tomorrow's ONE task
- [ ] Review your weekly schedule
- [ ] Delete ONE old/unnecessary file
- [ ] Check equipment battery levels

---

## MOTIVATION (5 min each)

- [ ] Read your "why I'm doing this" note
- [ ] Look at your progress visualization
- [ ] Listen to 5 minutes of a completed episode
- [ ] Read ONE section of ADHD-EXECUTION-GUIDE.md
- [ ] Write down ONE thing that's going well

---

## THE POWER OF 5 MINUTES

**On days when you can't focus:**
- These tasks prevent zero days
- Build/maintain momentum
- Keep you connected to the project
- Count as podcast work for streak
- Earn XP (5-15 XP each)

**Strategy:**
- Do just ONE
- If that feels good, do another
- If not, that's enough for today
- Tomorrow is a new day

---

**Pick ONE task above. Do it. Check it off. Done.**

**YOU DID SOMETHING TODAY. THAT'S A WIN.**
