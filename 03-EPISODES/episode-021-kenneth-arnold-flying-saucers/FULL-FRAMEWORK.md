### THE MISIDENTIFICATION CANDIDATES

**Explanation 1: LENTICULAR CLOUDS**

**What They Are:**
- Cloud formations near mountains (caused by wind over peaks)
- **Lens-shaped** (disc-like appearance)
- Often bright, reflective
- Can appear stationary OR moving (depending on wind)

**How They Explain Arnold's Sighting:**
- Mount Rainier = perfect location for lenticular clouds
- **Bright, shiny appearance** matches Arnold's description
- Multiple clouds in formation = possible

**Speed Problem Resolved:**
- Arnold's speed calculation ASSUMES objects traveled 47 miles
- **Reality:** Optical illusion (clouds closer than Arnold thought)
- If clouds were 5 miles away (not 25), speed calculation collapses

**Skeptic Point:**
"Arnold's distance estimate was GUESS. Wrong distance = wrong speed."

**Explanation 2: SNOW/ICE REFLECTIONS**

**The Theory:**
- Arnold saw sunlight reflecting off snow patches on mountains
- **Mount Rainier = glacier-covered**
- Bright flashes as sun angle changes

**Problems With This:**
- Doesn't explain formation flight
- Doesn't explain movement between peaks

**Explanation 3: SECRET MILITARY AIRCRAFT**

**Possibility:**
- U.S. military testing experimental jets/rockets
- 1947: Early jet age, secret projects common
- **Arnold's sighting near restricted military airspace?**

**Evidence:**
- No declassified projects match description
- But: Some projects stayed classified for decades

**Skeptic Hedge:**
