# WEEK 2: YOUR FIRST EPISODE
## Actually Recording a Full Episode

**Goal:** Record and complete ONE full episode
**Time:** 8-12 hours total
**Outcome:** Your first published episode!

---

## MONDAY: Choose & Prep (1 hour)

### Pick Your Episode:
1. Go to: `06-QUICK-WINS/easiest-episodes-first.md`
2. Pick ONE from "Easiest 10" (recommend: War of the Worlds or Kelly-Hopkinsville)
3. Read the EPISODE-BRIEF.md (5 min)
4. Read the RECORDING-NOTES.md (15 min)
5. Feel excited!

**Checklist:**
- [ ] Episode chosen
- [ ] Brief read
- [ ] Recording notes read
- [ ] Know what you're recording

---

## TUESDAY-WEDNESDAY: Deep Prep (2-3 hours)

### Preparation Work:
- [ ] Re-read RECORDING-NOTES.md
- [ ] Skim FULL-FRAMEWORK.md (just skim, don't read every word!)
- [ ] Mark up recording notes with your ideas
- [ ] Practice difficult pronunciations
- [ ] Set up recording space
- [ ] Test equipment

**Don't:**
- Read the full framework word-for-word (you'll reference it while recording)
- Overthink it
- Perfectionist paralysis

**Do:**
- Get familiar with the content
- Mark key sections
- Feel prepared enough

---

## THURSDAY-FRIDAY: Record! (3-4 hours)

### Recording Sessions:

**Thursday Session (1.5-2 hours):**
- [ ] Set up: mic, headphones, software, water
- [ ] Open FULL-FRAMEWORK.md for reference
- [ ] Record cold open (multiple takes OK!)
- [ ] Take 5-min break
- [ ] Record believer section (reference framework)
- [ ] STOP - save everything

**Friday Session (1.5-2 hours):**
- [ ] Record skeptic section
- [ ] Record your take section
- [ ] Record outro
- [ ] Save all files
- [ ] Name them clearly

**Tips:**
- If you make a mistake, pause, then re-say the sentence
- Don't restart entire sections
- Stay hydrated
- Take breaks
- It won't be perfect - that's OK

---

## SATURDAY: Edit (2-3 hours)

### Editing Process:
- [ ] Open all recordings in your editor
- [ ] Arrange sections in order
- [ ] Cut out long pauses (keep natural rhythm)
- [ ] Remove obvious mistakes
- [ ] Don't over-edit!
- [ ] Apply: Noise Reduction (if needed)
- [ ] Apply: Compression
- [ ] Apply: EQ (optional)
- [ ] Normalize to -16 LUFS
- [ ] Export as MP3

**Remember:**
- Done > perfect
- This is Episode 1, it's supposed to be rough
- You can always re-record later
- Get it done

---

## SUNDAY: Finish & Celebrate (1 hour)

### Final Steps:
- [ ] Listen to the full episode
- [ ] Note what to improve (for next time)
- [ ] Update COMPLETION-TRACKER.md
- [ ] Mark episode as COMPLETE
- [ ] Calculate XP earned
- [ ] CELEBRATE!

**You just recorded your first episode!**
